## About Me

I'm an **AI & Data Science engineering student** who got tired of only *reading* about infrastructure and decided to run it myself. That instinct turned into **CloudNest** — a self-hosted personal cloud running on bare-metal Ubuntu, and it's the reason my direction has shifted from "just AI/ML" toward **Cloud & DevOps**, where I get to combine the two: deploying, monitoring, and operating the systems that AI/ML workloads actually run on.

```yaml
Currently:
  degree: "B.E. AI & Data Science"
  building: "CloudNest — self-hosted infra on Ubuntu Server"
  learning: ["Kubernetes", "CI/CD pipelines", "Cloud (AWS/GCP)", "Terraform"]
  interests: ["Self-hosting", "Privacy-first tooling", "MLOps", "Observability"]
mission: >
  Learn infrastructure by running it — not just deploying to
  someone else's cloud, but understanding what's underneath it.
```

**Elevator pitch:** AI & Data Science student who self-hosts his own cloud instead of trusting Big Tech with his files — now applying that same hands-on approach to learning Cloud & DevOps engineering.
